--------------------------------------------------------------------------------
الجزء أ — تنسيق متقدم للنص (index.html + style.css)
--------------------------------------------------------------------------------

<!DOCTYPE html>
<html lang="en">
  <head>
    <meta charset="UTF-8" />
    <meta name="viewport" content="width=device-width, initial-scale=1.0" />
    <title>Text Formatting Lesson</title>
    <link rel="stylesheet" href="style.css" />
  </head>
  <body>
    <h1 class="myclass">hello world</h1>
    <h1 id="myid">welcome to my website</h1>

    <p class="p1">هذا النص مثال على الخط العريض (Bold) والمحاذاة.</p>
    <p class="p2">هذا النص مثال على الكلام المشطوب (Line-through).</p>
  </body>
</html>

/* style.css — الجزء أ */
body {
  background-color: #f4f4f4;
  font-family: Arial, sans-serif;
  padding: 20px;
  text-align: center;
}

.myclass {
  color: #e74c3c;
  text-transform: uppercase;
  text-decoration: underline;
  font-family: Impact, sans-serif;
  letter-spacing: 2px;
}

#myid {
  color: #2980b9;
  text-transform: capitalize;
  font-style: italic;
  text-decoration: overline wavy;
  text-decoration-color: #333;
}

.p1 {
  color: darkblue;
  font-size: 24px;
  font-weight: bold;
  text-align: right;
  direction: rtl;
  border: 1px dashed blue;
  padding: 10px;
}

.p2 {
  color: #333;
  font-size: 22px;
  text-decoration: line-through;
  text-align: center;
}

--------------------------------------------------------------------------------
الجزء ب — مثال بملف style1.css (خط مزدوج، تحويل لأحرف كبيرة، إلخ)
--------------------------------------------------------------------------------

<!DOCTYPE html>
<html lang="en">
  <head>
    <meta charset="UTF-8" />
    <meta name="viewport" content="width=device-width, initial-scale=1.0" />
    <title>تطبيق المحاضرة 28</title>
    <link rel="stylesheet" href="style1.css" />
  </head>
  <body>
    <h1 class="myclass">heading 1</h1>

    <p>هذا النص تجريبي لنرى الفرق بين العنوان والنصوص العادية.</p>
  </body>
</html>

/* style1.css — الجزء ب */
body {
  background-color: #f9f9f9;
  padding: 50px;
}

.myclass {
  text-align: center;
  text-decoration: double underline;
  text-decoration-color: red;
  text-transform: uppercase;
  font-family: Impact, sans-serif;
  font-size: 50px;
  font-style: italic;
  font-weight: bold;
  color: #333;
}
