### السبب الرئيسي

المساعد (GitHub Copilot coding agent) يعمل على مستودع **mahmoudshaabo1984/ahmed-sami-web-course** فقط — أي المستودع الذي يملكه الحساب نفسه. أما الـ Pull Requests التي أردت إضافة تعليقات عليها فهي موجودة في مستودع **خارجي** تابع لمنظمة أخرى:

- `CodeYourFuture/Module-Data-Groups/pull/1014` (Sprint 1)

المساعد **لا يملك صلاحية الكتابة (write access)** على مستودعات خارجية لا يملكها الحساب، لذلك لا يستطيع إضافة تعليقات أو أي تعديلات هناك.
